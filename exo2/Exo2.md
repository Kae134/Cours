# Bienvenu dans l'antre de l'exo 2

Apres avoir brave tout les dangers lors de la realisation de cette card profile, vous vous retrouvez
face au mechant formulaire et ses inputs, buttons et selects !

Bref tu vas juste faire un exo sur un form.

## Objectif
Faire un formulaire pour un celebre site connu pour son poisson qui demande : CA BULLE ?
Le formulaire doit permettre aux utilisateurs de saisir leurs informations personnelles et de s'inscrire à une newsletter.

### Champs à Inclure 
    - Nom : Un champ de texte pour le nom de l'utilisateur.
    - Prénom : Un champ de texte pour le prénom de l'utilisateur.
    - Email : Un champ pour saisir l'adresse e-mail de l'utilisateur (type email).
    - Mot de passe : Un champ pour saisir un mot de passe (type password).
    - Âge : Un champ pour saisir l'âge de l'utilisateur (type number).
    - Genre : Deux boutons radio pour sélectionner le genre (Homme ou Femme).
    - Pays : Une liste déroulante (<select>) pour choisir le pays de résidence.
    - Newsletter : Une case à cocher pour s'inscrire à la newsletter.
    - Télécharger un fichier : Un champ pour permettre à l'utilisateur de télécharger un fichier (type file).

### Bonus
Fait moi un style cool pour le formulaire (m'en fou si tu le trouves moche)




## Indices si tu bloques sur la facon de faire (en gros si tu veux pas utiliser google ;) )
En gros ce sont les bonnes pratiques a prendre.

### Structure du Formulaire :
    - Le formulaire doit être encapsulé dans une balise <form>.
    - Utilisez l'attribut action="#" pour le moment, car nous ne traiterons pas les données.


### Boutons :
    - Un bouton de type submit pour soumettre le formulaire.
    - Un bouton de type button qui affiche un message d'alerte (vous pouvez simplement mettre un texte descriptif pour le moment, car nous ne traiterons pas les événements).

### Mise en Forme :
    - Utilisez des balises <label> pour chaque champ afin d'améliorer l'accessibilité.
    - Ajoutez des sauts de ligne (<br>) entre les champs pour une meilleure lisibilité.
